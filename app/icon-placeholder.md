# Icon Placeholder Instructions

To complete the PWA setup, you need to create two icon files:

1. `/public/icon-192x192.png` - 192x192 pixels
2. `/public/icon-512x512.png` - 512x512 pixels

You can:
- Use an online icon generator
- Create simple colored squares with text
- Use a design tool like Figma or Photoshop

For a quick placeholder, you can create simple PNG files with:
- White background (#FFFFFF)
- Black text "WS" or "WSP" centered
- Or use a simple geometric shape

Once created, the PWA will be fully functional and installable.
